Développement d'une application mobile « MoveAndMe » avec framework React Native.

J’ai réalisé mon stage dans une société "ID Informatique", spécialisé dans le service du numérique. En ce moment la société travaillait sur une application mobile, qui scanne le QR code et stocke les données de QR code de clients. Le framework principal pour le développement de cette application est React Native. Je n'avais pas de cours sur React Native basé sur JavaScript à l'école. C'est pourquoi mon tuteur m'a donné le cours de React Native sur le site Openclassrooms.com. Ce cours est très riche. Il explique comment installer et configurer le React Native. Egalement, j'y ai appris de développer l’application pour récupérer les donnés à l'aide d'API pour afficher les annonces des films du site www.themoviedb.org

Installation et analyse du projet de l’entreprise « Scan du QRCode »

Cette application mobile est destinée à scanner le QR code et vérifier s'il est fonctionnel ou non. S'il est fonctionnel, l'application sauvegarde les informations qui se trouvent dans le QR code. S'il n'est pas fonctionnel, l'application affiche, que le QR code n'existe pas.

J'ai testé cette application sur mon ordinateur personnel.
