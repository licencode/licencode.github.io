Développer une application mobile avec framework React Native « MoveAndMe », l’application pour voir informations des films, développement par le cours de openclassrooms.com.

J’ai réalisé mon stage dans une société ID Informatique qui a domaine service du numérique. En temps de mon stage dans société était projet principale est une application mobile qui scanne le QR code et stocke les données de QR code et ses clientèles. En général l’application développée par React Native. Mais comme en mes études pas de cours de React Native qui écrit en JavaScript, mon tuteur m’a donné le cours de React Native dans Openclassrooms.com. Ce cours très riche il explique comment installer et configurer le React Native, et developper l’application pour prend les donnés par API et s’affiche les annonces des films du site www.themoviedb.org
